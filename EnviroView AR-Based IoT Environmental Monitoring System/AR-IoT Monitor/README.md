Contents:
- Assets: Contains the necessary project resources such as scripts, 3D models, textures, materials, scenes and files used in the Unity-Android application (AIoT-Monitor).
- Packages: Manages Unity package dependencies required for the project, including AR, UI and other relevant libraries.
- ProjectSettings: Configuration files responsible to define the project's Unity settings, such as input, build and platform configurations.

Note: This is not the whole Unity project, only files and directories relevant to the implementation of the system are included. Please download Unity along with the required libraries and packages to successfully open and run the project.
