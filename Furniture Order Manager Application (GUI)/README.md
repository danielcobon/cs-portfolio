Demonstrates the implementation of an application that handles orders with a GUI, a project report is also written to explain the project in further detail (includes asset files used for display)

- This java project is created in JDK13 on NetBeans IDE 12.6
- The 'c2650218_app_dev_ica2' folder contains the program for this assignment
- Under the 'Data' folder, empty text files called orderlist.txt and temporary.txt are included for the use of the program
- Under the 'Images' folder, images are provided for the use of the program
- 'Place Order' and 'Review Order' have been combined into only 'Place Order' in this program as to make it more user friendly (It does both operatoins at once, explained in the report)
