Demonstrates the ability to analyze an existing program and implement object-oriented programming principles, appropriate software design patterns, an additional game mode, and a simple chat room.

Language: Python
IDE: NetBeans

Includes:
- A video explanation analyzing the program and the appropriate software design patterns to be implemented (with accompanying diagrams).
- Source code for the "Pocket Beasts" game, broken down into phases:
Phase 1: Implementation of suitable software design patterns
Phase 2: Creation of an additional game mode
Phase 3: Integration of a chat room feature
- A report detailing the actions taken based on design feedback, the completeness of the solution, and the advantages and limitations of object-oriented design patterns.
