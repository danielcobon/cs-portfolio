Demonstrates the ability to work in a group using Agile development (SCRUM) to complete a diary application with online security (Firebase) and phone sensor (location) integration. Users can create entries with optional location and photos, all stored on Firebase. The application also displays statistical data, such as the percentage of a user’s moods across entries, visualized with a pie chart.

Language: Kotlin
IDE: Visual Studio
Tools: Jetpack Compose, Firebase

Includes:
- Diary App: Application program source code, with all visual assets and supporting components used included.
- Documentation: Records of each SCRUM cycle carried out throughout the project’s lifecycle. Also includes visual designs, diagrams and video demos of the project.
- Reflective Report: Personal reflective report detailing my contributions to the application, the group’s dynamics in an Agile development environment, and an in-depth analysis of my personal and professional growth following the completion of the project.
